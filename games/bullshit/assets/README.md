# Bullshit Visual Assets

The two source PNG files in this directory are original generated artwork created for LAN Party Hub on 2026-07-23 with OpenAI's built-in image generation tool, then downscaled with nearest-neighbour sampling. Matching WebP files are browser-encoded runtime derivatives that reduce first-load transfer size without changing the art direction.

- `pixel-wood-table.png`: top-down warm pixel-art wooden table, generated without cards, UI, logos, text, or third-party references.
- `pixel-card-back.png`: original symmetrical indigo/coral/gold pixel-art card back, generated without logos, suit marks, text, or third-party references.
- `pixel-wood-table.webp` and `pixel-card-back.webp`: runtime derivatives of the corresponding PNG sources.

The prompts explicitly excluded Balatro artwork, casino-felt styling, recognizable branded motifs, and copied card designs. These project-authored assets are distributed under the same Apache-2.0 license as the `@open-party-lab/game-bullshit` package.
